# meeting-prep

会议准备工具，通过 6 个结构化问题引导用户整理出完整的会前文档，最终保存为 Obsidian 笔记。

## 使用方式

在 Claude Code 中输入 `/meeting-prep` 或说"帮我准备会议"即可触发。

## 问答流程

工具会逐一提问以下 6 个问题（每次只问一个）：

1. **会议目的** — 做决策 / 同步信息 / 解决问题 / 头脑风暴
2. **参会人与角色** — 决策者、汇报者、执行者
3. **议程条目** — 每条议题及时间分配
4. **预读材料** — 参会前需要看的文件或背景信息
5. **期望产出** — 会议结束时要拿到的结论或成果
6. **时间与地点** — 会议时间、时长、线上/线下

完成后自动生成格式化的会前文档，保存到 Obsidian vault。

## 安装

```
/plugins install meeting-prep@henry-hub
```
