# 项目案例

> **[需更新]** 以下案例来自历史投标文件，新项目投标时需核实案例状态、补充最新案例，并按招标文件要求提供合同复印件（合同首页、标的及金额所在页、签字盖章页、签署日期页）和验收证明。

## 一、案例格式模板

每个案例应包含以下要素：

```
### 案例名称：{项目名称}
- **客户名称**：{甲方全称}
- **合同金额**：{金额}万元
- **签订时间**：{年份}
- **项目状态**：{结项/在建}
- **项目内容**：{简要描述项目范围和主要建设内容}
- **项目成果**：{交付成果和客户评价}
- **联系人**：{姓名}，{电话}
- **备注**：{是否包含信创环境、是否可作为特定加分项等}
```

## 三、案例选择策略

1. **数量要求**：一般要求近三年至少2-3个类似案例；部分项目提供越多得分越高（如国家制造业项目：3项得3分，每增加1项得0.5分，最多4分）。
2. **金额门槛**：注意招标文件对合同金额的要求（如"不低于100万"）。
3. **信创加分**：包含信创环境下办公系统建设案例可获额外加分。
4. **证明材料**：需提供合同关键页复印件（合同首页、标的及金额页、签字盖章页、签署日期页）及验收证明复印件。
5. **时间范围**：注意招标文件规定的时间窗口（以合同签订时间为准）。

## 四、行业服务范围

璞华科技典型案例覆盖以下领域：
- **数字金融**：股权投资管理、基金管理、债权融资管理
- **智能智造**：产品生命周期管理（PLM）、设备智能维保
- **智慧教育**：精准教学、AI 自动批阅
- **政府投资基金**：基金管理系统、投资监管平台
