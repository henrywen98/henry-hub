# 统一门户与工作台

## 概述

统一门户与工作台是投资管理信息系统的统一入口和信息展示平台，承担着打通各业务系统间信息对接、实现一站式办公协同的核心职责。该模块根据用户角色（领导、投管、风控、财务等）动态展示差异化的首页看板、关键指标、待办事项和快捷入口，支持集团与子公司之间的业务及数据逻辑隔离，同时提供PC端和移动端的全终端覆盖能力，确保业务人员随时随地高效办公。

## 详细功能

### 多角色门户

#### 功能说明

构建统一的应用门户入口，用户通过用户名、密码、机构编号登录系统。登录成功后，系统按照用户角色动态展示个性化首页内容，包括领导主页、业务人员主页等不同视图。系统通过机构编号实现集团与子公司之间的业务及数据库逻辑隔离。

#### 核心操作流程

1. 用户输入凭证登录系统，系统识别用户角色和所属机构
2. 根据角色动态加载对应的首页看板布局
3. 展示该角色权限范围内的关键指标、项目视图、汇总台账等内容
4. 用户可自定义配置首页模块和关注指标

#### 关键特性

- **关键指标展示**：汇总展示项目数量、累计投资金额、基金总实缴金额、基金总认缴金额、投资回款金额、累计投资收益、基金总估值等核心指标，点击指标名称可查看组成明细，支持不同人员自定义灵活配置关注的指标
- **项目视图**：提供对投资项目的整体概览，包括投资金额、投资时间、投资类型、投资阶段、持股比例等基本信息以及财务数据、业务数据等详细信息，点击具体项目可查看明细数据
- **汇总台账**：根据每个人的数据权限查看权限范围内的多种报表台账和图表，如负责的项目列表、行业分布、地域分布等
- **快捷入口**：支持用户按照自定义需求配置个人快捷入口，如快速记录项目跟踪、项目分红、添加项目、添加任务等

### 流程工作台

#### 功能说明

为业务人员提供统一的流程发起与管理平台，汇聚包括股权投资、基金管理、办公协同等各类业务的待办信息，便于统一审批处理，提升审批效率。

#### 核心操作流程

1. 业务人员进入流程工作台查看可发起的流程类型
2. 选择目标流程（项目流程、基金流程或日常办公流程）发起申请
3. 系统自动将流程推送至相关审批节点
4. 审批人在工作台处理待办事项

#### 关键特性

- **申请入口**：为发起项目、基金流程提供便利视图，可汇总查看任意一个项目或基金的全周期流程，支持业务快速发起项目全周期流程；同时提供用印、事项、出差等日常办公流程申请入口
- **委托待办**：业务人员休假期间可将待办任务事项委托他人办理，支持设置委托有效时间，可随时关闭委托待办。分类展示委托待办的各类流程，审批中流程展示任务主题、当前审批人、当前节点等信息
- **统一审批**：汇聚各业务系统的待办信息，支持通过关键词对待办事项、已办事项、我的发起进行筛选，实现各业务审批的统一入口

### 个人中心

#### 功能说明

个人中心汇总展示当前登录用户的流程、事项、任务、提醒等各类信息列表，方便用户集中管理个人事务。

#### 核心操作流程

1. 用户进入个人中心查看待处理事项
2. 对待办任务进行审批、转签、加签等操作
3. 查看已办任务历史记录和通知消息
4. 对流程进行撤回、补充意见等后续操作

#### 关键特性

- **待办任务**：汇总所有待办任务，支持查看详情和审批，对加急待办有加急标志。支持前后加签、转签，可灵活设置抄送人员并自动发送提醒
- **我的已办**：汇总所有已办任务，支持补充意见、传阅，以及对未审批完毕的任务进行撤回操作。业务人员可查看历史已处理流程的完整信息
- **通知消息**：汇总展示所有通知消息，支持一键已读功能，并支持对消息发起的来源用户进行回复及讨论

### 自动提醒

#### 功能说明

提供灵活的事项提醒配置机制，支持日常提醒、项目提醒、基金提醒等多种事项类型，覆盖有一定规律的日常工作场景。

#### 关键特性

- 支持配置提醒对象（用户/角色）、提醒方式（系统通知/邮件/APP推送）、提醒频率（定时/周期）等条件
- 灵活适配各种提醒触发场景，如基金call款提醒、投后报告提交提醒、合同到期提醒等

### 移动端门户

#### 功能说明

按照集团统一规划与设计要求开发移动端应用，功能覆盖投资系统的待办审批、项目/基金查询、文档阅览、消息提醒等核心功能，支持手机端和平板端展示。

#### 核心操作流程

1. 用户通过移动端登录，进入移动首页查看功能指引和快捷入口
2. 在线审批各类业务流程，处理待办事项
3. 查询项目和基金的详细信息
4. 接收和查看系统推送的消息通知

#### 关键特性

- **移动端首页**：包含查企业、基金、项目、子基金、工作进展等功能指引，支持通过关键词进行信息搜索，展示常用功能快捷入口和行业资讯
- **流程审批**：支持基金管理、项目投资等业务流程在移动端审批，展示待办、已办、我的申请事项，支持同意、退回等操作
- **消息提醒**：系统中业务流程在对应节点触发的通知消息支持移动端查看
- **通讯录**：支持组织架构及人员通讯录查看，展示人员姓名、部门、职位、邮箱、联系方式等
- **基金管理**：展示管理的基金名称、出资主体、基金经理、状态、规模、认缴金额，详情页包含基金规模统计、项目成员、项目概况、工商信息、投资信息、现金流、文件库等功能
- **项目管理**：项目列表分为项目维度和企业维度，支持按管理人、项目状态、出资主体、基金、地域、行业等多维度筛选，可查看投前和投后详情
- **文档阅览**：支持在移动端查看基金或项目管理过程中的附件，支持在线预览附件详情

### 移动端业务门户

移动端门户根据用户角色展示差异化的业务数据视图：

- **管理人视图**：展示工商信息、人员结构、经营计划、管理基金列表、填报统计、合同签署情况、文件库
- **基金卡片**：基金简称、企业全称、基金编号、基金状态、设立日期、项目数、认缴规模；详情页含工商信息、基金信息、认缴信息、管理团队、投资信息、定期估值、基金业绩等Tab页
- **直投项目视图**：支持项目维度和企业维度双列表切换；卡片展示项目名称、项目状态、出资主体、项目经理、首次出资日期、累计投资、投资占比
- **子基金视图**：子基金名称、出资主体、基金经理、子基金状态、子基金规模、认缴金额

### 组织认证与权限管理

基于Spring Security + OAuth2 + JWT实现SSO单点登录方案。采用RBAC（基于角色的访问控制）权限模型，支持多维度权限配置（功能权限、数据权限、字段权限）。系统安全审计记录所有用户的关键操作日志，含操作时间、操作人、操作类型、操作对象、操作结果等信息。
