# 股权投资管理

## 概述

股权投资管理是投资管理信息系统中最核心的功能模块，覆盖股权投资从投前、投中到投后的全生命周期管理。通过该模块实现投资计划的下达与执行、项目库的全流程管理、控股股权投资（直投）的差异化流程配置，以及投后阶段的一企一档式管理。模块支持投资计划管理、项目申报-立项-尽调-投决-合同-交割-投后-退出的完整业务闭环，帮助投资机构实现业务流程标准化、数据资产沉淀和投资决策数据化。

系统支持对集团及二级平台公司的多种投资业态进行需求适配，区分财务投资和战略投资等类型，并通过灵活的流程配置引擎实现不同组织、不同业务类型的差异化流程管理，满足从大型国有投资平台到市场化基金管理人的多层级管理需求。

## 详细功能

### 投资计划管理

#### 功能说明

通过投资计划管理模块，实现股权投资和基金投资计划的下达、填报、审核、变更、执行、历史记录查看全流程线上化管理，以及对应的资金计划，同时增强投资计划与项目库的数据联动管理能力，支持计划落实进度在线查看。

#### 核心操作流程

1. 任务管理员自定义计划任务发送，配置任务标题、下发时间、填报截止时间、任务发送范围
2. 各子公司通过投资计划报送模块填写年度计划（区分基金/股权计划），包含项目名称、类型、持股比例、金额、回报率、资金来源、季度目标等
3. 各级经办人对所辖公司的填报计划汇总提交，系统支持投资计划汇总逐级审核
4. 审批后的计划通过变更窗口期进行调整，系统自动对修改、删除和新增进行标记
5. 系统每月定期发送填报任务，项目负责人填报各投资计划项目实际执行情况

#### 关键特性

- **投资任务下发**：用户可新增任务，配置任务标题、下发时间、截止时间、发送范围，系统按配置规则发送
- **投资计划填报**：区分基金/股权计划，字段含项目名称、类型、持股比例、金额、回报率、资金来源、季度目标等
- **投资计划变更**：窗口期内提交变更申请，系统自动带出最新数据，标记修改、删除和新增内容，审批通过后生效并对历史数据进行留痕
- **投资计划执行**：系统自动将年度计划与项目关联，联动查看投资阶段和进展情况，超预算、进度滞后自动预警，并根据工商统一社会信用代码判断各子公司投资计划是否重叠
- **投资进度月度上报**：系统每月定期发送填报任务，项目负责人填报实际执行情况、本月投资金额及累计投资金额
- **投资计划备案**：审核通过的投资计划汇总上报至国资委备案，支持备案文件留档
- **智能汇总分析**：自动将年度投资计划与执行情况数据进行汇总分析，通过可视化图表动态呈现，支持按主业/非主业、新建/结转等维度生成图表报告

### 项目库管理

#### 功能说明

项目库是所有投资项目的统一管理入口，覆盖从企业信息维护、项目入库、跟踪走访到退出、终止和清算的全流程管理，确保项目信息的完整性和可追溯性。

#### 核心操作流程

1. 维护潜在项目企业信息（含已注册和未注册企业、境内外企业），支持关联企查查查询工商注册信息
2. 项目入库申请，补录项目情况、所属阶段、投资亮点等信息，发起入库审批流程
3. 定期录入项目跟进走访记录，进行线上项目跟踪落实
4. 根据项目状态变化，自动或手动将项目归类至退出库、终止库或清算库

#### 关键特性

- **企业维护**：实现潜在项目企业信息维护，针对已工商注册企业提供关联企查查查询带出注册信息的功能
- **项目入库**：支持按集团及各子公司管理要求进行入库流程配置，流程节点包括发起人、部门负责人、风险管理部门、战略发展部门、分管领导、总经理等多级审核
- **项目走访跟进**：支持录入项目跟进走访记录，支持查看项目历史跟踪记录
- **项目退出库**：已完全退出的项目自动转入，可查看项目全生命周期过程中的所有审批、事项、材料和数据
- **项目终止库**：投前决策过程中项目经理主动发起终止申请，审批通过后自动从投前项目库转入
- **项目清算库**：对非主业企业、非优势企业、非正常经营企业等由项目经理手动标记，跟踪集团内部决策、清理备案、清算审计、司法清算、税务注销、工商注销等全过程

### 投前管理

#### 功能说明

投前管理覆盖项目申报、立项、尽调、跟踪、方案审查、会议管理、合同管理和风险管理等环节，支持差异化流程配置，确保投资决策过程规范、可追溯。

#### 核心操作流程

1. **项目申报**：在项目池中录入申报信息，包含企业工商信息、项目相关信息（项目名称、投资轮次、所属阶段、项目经理、出资主体、项目简介等）、业务团队组织信息
2. **项目立项**：维护立项信息（被投企业融资轮次、投资角色、融资金额、投前/投后估值、拟投金额、投资占比等），支持核验资料上传完成性后开启下一步流程
3. **项目尽调**：维护尽调信息，包括计划名称、尽调范围、尽调小组成员、起止时间、尽调类型、尽调机构信息、访谈记录、三方尽调报告等
4. **项目跟踪**：投资经理通过PC端和移动端记录项目跟踪信息，跟踪记录可实时反馈给相关领导，支持消息在线回复和交流
5. **方案审查**：对被投企业进行投资方案审查（拟投资金额、拟持股比例、投资亮点、价值判断、盈利预测），风险评审（投前预测风险和应对方案），主要协议条款评审
6. **投资决策**：管理投资决策信息，实现投委会在线投票，包括融资轮次、融资金额、出资主体、本轮投资金额、投资占比、决策材料、投决结果等
7. **合同管理**：发起投资合同审批流程，包括合同名称、合同编号、重点关注条款、合同附件等，审批完成后展示关键条款变更之处

#### 关键特性

- **会议管理**：维护总办会、投委会等各类会议记录，包括会议主题、时间、类型、参与人员、会议内容和会议纪要附件
- **风险管理**：维护被投项目风险评级信息，包括评级时间、级别及支撑材料
- 支持AI辅助功能，如BP智能解析、尽调报告生成等

### 投中管理

#### 功能说明

投中管理覆盖投资主体权益管理、合同管理、核心条款提取、出资审查、项目划款和股权交割等环节，确保从投资决策到资金支付、法律确权的完整闭环。

#### 核心操作流程

1. 系统自动沉淀汇总各投资主体的权益指标信息
2. 管理项目全生命周期过程中的投资合同，沉淀每个投资主体的交易数据
3. 在合同审查通过后进行项目出资，按协议进行出资划款
4. 投资款划付后完成工商变更和股权交割确权

#### 关键特性

- **投资主体**：自动通过投资协议、投资划款、股权变更、估值和回款等模块的数据进行主体权益计算，指标包括协议金额、签约日期、出资金额、最新持股比例、已回款金额、持有股份最新估值和最新IRR等。系统自动记录每次权益变更记录，支持查看历史任意时点的持股权益数据
- **项目合同**：查看合同主要信息和审批记录，沉淀每个投资主体的投资估值、持股比例、持股数量、出资金额、核心权利条款
- **核心条款**：统一汇总项目合同中的重要交易条款，支持AI大模型自动读取投资协议中的核心条款，包括业绩对赌、大股东回购、反摊薄权、赔偿机制、优先购买权等。对需定期检查的关键条款支持设置提醒
- **出资审查**：涵盖关联事项、项目合同、关键条款、项目划款等方面，支持保证金与投资款之间的转化。投资经理发起划款审批流程，记录预计缴款时间、预计出资主体、预计缴款金额等，支持上传银行汇款单等印证材料
- **项目划款**：自动汇总项目全生命周期流程提交的所有划款申请，付款信息包括付款申请人、申请时间、关联投资合同、付款主体信息、付款金额、币种和收款方信息
- **股权交割**：录入协议条款和交割时间节点，对接消息提醒功能提醒逾期未完成交割，上传营业执照、批复文件等确权材料，同步至资产监控平台完成国有资产登记

### 投后管理

#### 功能说明

投后阶段基于企业维度构建一企一档，对接天眼查、企查查等外部数据源获取企业工商信息，覆盖工商信息更新、出资权益、法人治理、三会管理、估值、财务分析、关键条款跟踪、退出执行、清算注销等核心环节。项目保存后系统自动后台获取企业工商数据、股东信息、董监高人员、变更记录、异常经营、行政处罚、法律诉讼、失信行为、法院公告、新闻舆情、子公司信息和专利信息等。

#### 核心操作流程

1. 项目进入投后阶段，系统自动创建企业档案并获取外部工商数据
2. 投后团队定期跟踪维护出资权益、三会管理、法人治理等信息
3. 项目经理定期收集并上传被投企业财务报表和估值数据
4. 系统自动汇总现金流、计算IRR/DPI/MOIC等关键指标
5. 根据退出条件触发退出决策流程

#### 关键特性

- **出资权益**：自动沉淀汇总各投资主体的权益指标信息，包括协议金额、签约日期、出资金额、最新持股比例、已回款金额、已回款本金、已回款收益、持有股份最新估值和最新IRR、DPI、MOIC等指标。系统自动记录每次权益变更
- **三会管理**：管理被投企业的董事会、股东大会和监事会，开会前对议案内容审批，重大事项自动同步到项目重大事项记录中，会议结束后归档议案结果信息
- **法人治理**：管理和记录外派到被投企业的董监高人员及任期，包含外派人员姓名、职位、时间、地点和离开日期，支持人员变更审批流程管理，到期前三个月进行邮件和短信提醒
- **定期走访**：投后人员定期跟进目标企业运营情况，系统预设定期走访频率，超过频率未跟踪时自动对相关人员发出提醒
- **CapTable后轮融资情况**：被投企业发生新一轮融资或股权变更时，通过股权变更功能调整股权情况，审批通过后自动更新出资主体权益信息
- **投后报告**：项目经理定期上传投后管理报告，审批通过后自动归档。系统每季度自动提醒上传，并能按照报告模板自动抓取系统数据生成报告草稿
- **项目估值**：支持可比公司、可比交易、最新融资价格、市场乘数、市场价格等多种估值方法。上市公司可根据基准日期自动获取外部股票收盘价进行估值计算。多个出资主体时按持股情况自动进行估值计算，记录历史估值信息
- **财务数据**：支持被投企业财报Excel批量导入或PDF/图片的OCR智能识别服务，快速将财务数据导入系统并形成多年度对比分析
- **关键条款跟踪**：针对合同执行事项进行投后定期跟进，确认待落实事项的落实进度、当前状态、跟进处理人和后续推进，实时监控合同执行状态。条款类型包括交割后义务、付款条件、董监高委派、反稀释条款、对赌条款、回购条款、领售权条款、共同出售权、优先购买权等
- **项目现金流**：汇总所有出资划款、项目回款、分红和最新项目估值，形成整体现金流流水，自动计算IRR、DPI、MOIC等关键指标。每季度财务指标波动分析、与投决时指标年度差异分析，差异值达到阀值时启动差异预警分析
- **关注事项**：由风控人员对项目过程的关注事项进行检查提醒，支持事项反馈跟踪

### 差异化流程配置

#### 功能说明

全业务流程可通过系统配置进行灵活调整，实现不同组织、不同业务类型的差异化流程管理，确保流程适配多元化的管理模式。

#### 关键特性

- 项目投资阶段以流程推动，从项目储备、立项、尽调、投决、执行、投后到退出的全流程管理
- 支持配置每个阶段及节点的审批流程、审批业务表单、发起人角色、是否需要上会
- 支持配置合规性要求，包括需上传的附件、阶段自动流转、各环节依赖数据、是否进入投后阶段
- 预设流程包括：储备项目流程、立项审批流程、投资决策流程、协议签署流程、出资流程、国资备案流程、重要事项报告流程、退出决策流程、退出执行流程、投委会子流程、变更项目经理流程等

### 子公司认定

#### 功能说明

在系统内新设公司或公司发生重大信息变更时，根据子公司认定表模板自动生成认定表并支持发起业务审核。

#### 关键特性

- 股权二级或管理二级及以下公司发生对外新设/增资/并购/参股等经济行为时，由管理职能发起并经相关部门会签后，由决策机构履行内部决策程序
- 系统可根据设定条件严格控制管理三级次企业新设/并购子企业，严禁出现管理四级次企业

### 后评估报告

#### 功能说明

项目经理定期上传评估报告和评估结果性信息，系统每季度自动提醒上传，并能按照后评估报告模板自动抓取系统数据生成报告草稿。

#### 关键特性

- 支持报表及报告模板配置功能
- 支持在线查看模板，并带有水印功能，满足信息披露的合法合规要求
- 支持发起审批流程、在线修改、定稿版上传留档、添加水印

### 退出管理

#### 功能说明

支持转让、上市、清算等多方式退出决策与执行跟踪，确保退出过程规范、收益核算准确。

#### 核心操作流程

1. 投资经理或投后负责人发起退出决策申请
2. 填写拟退出时间、比例、金额、退出方式、交易对手、退出报告等信息
3. 根据退出决策流程提交申请及审批
4. 审批通过后进入退出执行阶段，跟踪退出进展直至完成

#### 关键特性

- **退出决策**：支持部分退出和全部退出，退出方式包括转让退出、上市退出、清算退出等，记录拟退出时间、比例、金额、交易对手等
- **退出执行**：维护退出信息包括退出时间、退出金额、是否已分配、分配金额、分配本基金金额、分配时间等，支持线上审批
- **文件库**：树形结构展示当前项目全部关联文件，覆盖立项报告、尽调报告、风控意见书、投决会会议纪要、投资协议、公司章程、投后管理报告、财务报表、退出方案等全生命周期文档，支持在线预览和下载

### 项目状态流转

直投项目完整状态流转：储备 → 已立项（发起立项审批通过）→ 已决策（发起投资决策审批通过）→ 协议已签署 → 已出资 → 部分退出/全部退出。特殊状态：储备阶段可直接跳转到协议已签署（适用于已有明确投资意向的项目）。已终止：储备至已决策阶段可设置终止。非正向情况：下一环节审批不通过时，状态不更新，停留在当前阶段。

### 入库业务数据字段

项目入库时需填写的完整业务数据：
- **基本信息**：运营主体所在地、注册状态、是否敏感项目、企业代号、企业简介
- **股权结构表**：股东名称、持股比例、认缴金额、实缴金额
- **主要产品表**：产品名称、产品类型、市场份额、竞争优势
- **财务指标表**：近两年总资产、净资产、主营收入、净利润、经营现金流
- **上市情况**：上市板块、股票代码、上市状态
- **投资角色**：领投、共同投资、跟投、单一投资
- **融资轮次**：天使轮、Pre-A轮、A轮、A+轮、Pre-B轮、B轮、B+轮、C轮、C+轮、D轮、E轮及以后、PreIPO轮、IPO轮、定增、可转债、受让老股、二级市场买入
- **项目来源**：内部推荐、地方推荐、行业协会推荐、中介机构推荐、主动申报、其他

入库审批流程参考：发起人填写 → 主办部门审核 → 风险管理部审核 → 战略发展部审核 → 主办部门分管领导 → 风险管理部分管领导 → 战略发展部分管领导 → 总经理审批 → 结束（知会项目组成员）。业务规则：总经理批准后365天未立项的项目自动出库。

### 投前管理详细功能

- **投前立项**：包含拟投资金额、项目情况描述、尽调情况说明，提交审批后状态变更为"已立项"
- **项目尽调**：成立尽调小组，维护小组成员信息和分工，记录尽调结果和尽调报告
- **项目内核**：内部风险检查环节，由风控部门对项目进行风险评估
- **投资决策**：记录被投企业融资轮次、融资金额、出资主体、本轮投资金额、投资占比、投决结果；支持投票人选择和投票汇总（赞成/反对/弃权）
- **投前用印**：直接调用OA用印申请流程
- **协议签署**：分投资协议和其他协议两类，记录协议规模、签署日期、协议编号
- **投资支付**：记录出资主体、认缴规模、本期应付款、实付金额、付款时间、收款账号

### 投后管理详细功能

- **项目进展**：记录日常沟通事项（拜访、电话、邮件），支持进展评价
- **关注事项**：风控人员设置检查提醒项，投后经理定期反馈
- **重要条款跟踪**：投前投后条款统一展示，可对单个或全部主体进行条款维护
- **人员委派**：董事、监事派遣，直接调用人事管理委派流程
- **国资备案**：备案事项类型、备案时间、备案材料上传
- **融资事项**：记录再融资轮次、融资金额、对估值和股本股价的影响、出资方明细
- **三会事项**：董事会、监事会、股东会议案的线上审批流转
- **定期估值**：支持按最新融资价格、市场乘数、市场价格、成本法等多种方式计算估值
- **财务信息**：对接AI财报识别模块，自动提取并录入被投企业财务数据
- **退出决策与执行**：部分退出/全部退出，记录退出方式（IPO/并购/回购/转让等）、退出金额、分配金额

### 投后报告模板

"投后目标完成情况确认表"结构包含：项目基本信息（项目名称、投资日期、投资金额、持股比例）、应推动事项/中期投后目标、进度安排（完成百分比）、战略指标达成情况、财务指标达成情况（收入/利润/现金流vs计划值）、增值服务目标完成情况、项目整体投后风险评估、风险分类（正常类/关注类/次级类/损失类）。

### 委派代表履职管理

"投资项目履职情况表"设计包含：申请信息（委派人、委派企业、职务）、参会情况（会议名称、时间、主要内容、决议/表决意见）、参加其他活动情况、廉洁从业情况、重大事项报告概述。履职情况定期汇总形成履职报告，纳入绩效考核参考。
