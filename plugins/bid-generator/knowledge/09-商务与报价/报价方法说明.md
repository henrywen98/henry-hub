# 报价方法说明

## 一、常见评标方法与报价策略

### 1. 综合评分法（最常用）

综合评分法是政府采购和企业招标中最普遍采用的评标方式，投标价格只是评分要素之一，通常占总分的30%-40%权重，其余为技术方案、公司资质、服务能力等维度。

**价格分计算公式（常见方式）**：
- 方式一（基准价法）：价格分 = (评标基准价 / 投标价格) x 价格权重分值
  - 评标基准价 = 满足招标文件要求的所有有效投标价的算术平均值
- 方式二（最低价法）：价格分 = (最低投标价 / 投标价格) x 价格权重分值
- 方式三（固定基准价）：招标方设定最高限价或预算价，以此为基准计算

**报价策略**：
- 综合评分法下不应盲目压价，而应综合考量技术、服务、资质等各维度得分优势
- 报价应在保证利润的基础上，根据竞争对手预估报价合理定位
- 当技术分占比高（如60%以上）时，适当提高报价可接受；当价格分占比高（如40%以上）时需更关注价格竞争力

### 2. 最低评标价法

适用于标准化程度高、技术要求相对简单的项目，投标报价经评审修正后，最低价者中标。

**报价策略**：
- 在满足招标文件实质性要求的前提下，尽可能压低报价
- 注意不能低于成本价，否则可能被认定为异常低价或废标

### 3. 竞争性磋商/竞争性谈判

通过多轮磋商或谈判确定最终价格，适用于技术复杂或无法精确确定规格的项目。

**报价策略**：
- 首轮报价可适当留有议价空间
- 后续轮次根据磋商反馈逐步调整

## 二、报价编制要点

### 1. 总价与分项价格一致
投标人需确保投标书中的总报价、开标一览表中的价格、投标分项汇总表中各项合计金额完全一致。如按单价计算结果与总价不一致，以单价为准修正总价。

### 2. 全费用报价原则
投标人提交的项目报价应为合同执行中的全部费用，包括但不限于：
- 软件开发及定制化费用
- 系统安装、调试、部署、集成费用
- 项目实施人工费用
- 培训费用
- 差旅及驻场费用
- 项目管理费用
- 税费
- 其他为保证系统全面正常运行所需的资源费用

招标人不需额外向投标人支付任何费用，投标人应充分考虑隐性成本。

### 3. 避免异常报价
- 报价不应高于招标文件规定的最高限价（预算控制价）
- 报价不应明显低于成本，否则可能被要求提供成本说明或被认定为废标
- 若投标人不提供分项报价，将被视为没有实质性响应招标文件

### 4. 投标保证金
按招标文件要求缴纳投标保证金（如国家制造业项目要求30,000元），在投标有效期内撤回投标的，保证金将被没收。

**国家制造业项目报价**：总价285万元，服务期90天

| 模块 | 金额 |
|------|------|
| 办公协作管理 | 42万元 |
| 财务管理 | 85万元 |
| 投资管理 | 98万元 |
| 人力资源管理 | 30万元 |
| 监督检查 | 30万元 |
| **合计** | **285万元** |

> 投标时应根据项目实际需求、招标文件预算限制、竞争态势等综合因素确定报价策略。
