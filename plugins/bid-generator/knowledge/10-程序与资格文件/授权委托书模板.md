# 授权委托书模板

## 一、法定代表人授权书

法定代表人授权书是投标文件的必备程序性文件，用于证明投标代表人经法定代表人正式授权，有权代表公司参与投标活动并签署一切相关文件。

### 模板正文

```
法定代表人授权书

本授权委托书声明：我（{法定代表人姓名}）系注册于（{公司注册地址}）的（{公司全称}）的法定代表人，现代表公司授权下面签字的（{被授权人姓名}/{职务}）为我公司合法代理人，代表本公司参加{项目全称}（招标编号为{招标编号}）的投标活动。代理人在本次投标中所签署的一切文件和处理的一切有关事物，我公司均予承认。

本授权书于 {年} 年 {月} 月 {日} 日签字生效，特此声明。



法人代表：
（签字或盖章）

授权代表（被授权人）：
（签字或盖章）

单位名称（公章）：{公司全称}




后附法人代表和授权代表的身份证复印件加盖公章
```

## 二、附件要求

### 1. 法定代表人身份证
- 提供身份证正反面复印件
- 复印件上加盖公司公章
- 粘贴在授权书指定位置或作为附页

### 2. 被授权人（授权代表）身份证
- 提供身份证正反面复印件
- 复印件上加盖公司公章
- 粘贴在授权书指定位置或作为附页

## 三、填写说明

### 关键信息清单

| 信息项 | 说明 | 示例 |
|--------|------|------|
| 法定代表人姓名 | 营业执照上的法定代表人 | 李晓敏 |
| 公司注册地址 | 营业执照上的注册地址 | 中国（江苏）自由贸易试验区苏州片区苏州工业园区集贤街89号独墅湖数字经济产业园5幢1201室 |
| 公司全称 | 营业执照上的全称 | 璞华科技有限公司 |
| 被授权人姓名/职务 | 实际参与投标开标的人员 | 冷显明/客户经理 |
| 项目全称 | 招标文件中的项目名称 | 国家制造业转型升级基金股份有限公司投资管理信息系统采购项目 |
| 招标编号 | 招标文件中的编号 | ZC25T320067 |
| 授权日期 | 通常为投标文件签署日前1天 | 2025年7月27日 |

### 注意事项

1. **签章完整性**：法定代表人和被授权人均需签字或盖章，单位需加盖公章
2. **日期逻辑**：授权日期应早于或等于投标文件签署日期，且在投标截止日期之前
3. **授权范围**：授权书中应明确是针对本次特定项目投标活动的授权
4. **身份证有效期**：确保法定代表人和被授权人的身份证在投标日仍在有效期内
5. **公章一致性**：授权书上的公章应与营业执照上的单位名称完全一致
6. **公司更名情况**：如公司曾更名，需在投标文件中提供更名材料（工商变更通知书、更名公告等）

## 四、法定代表人直接投标

如果法定代表人本人直接参与投标活动，通常不需要单独的授权委托书，但部分招标文件仍要求提供法定代表人身份证明书。格式如下：

```
法定代表人身份证明书

单位名称：{公司全称}
地  址：{公司注册地址}
姓  名：{法定代表人姓名}
性  别：{性别}
年  龄：{年龄}
职  务：{职务}

系{公司全称}的法定代表人。
特此证明。


单位名称（公章）：{公司全称}
日期：{年} 年 {月} 月 {日} 日

附：法定代表人身份证正反面复印件（加盖公章）
```

> 具体格式以招标文件要求为准，部分招标文件会提供固定格式的授权书模板，应优先使用招标方提供的模板。
