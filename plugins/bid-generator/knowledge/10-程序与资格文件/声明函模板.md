# 声明函模板

## 一、关于投标文件的声明函

声明函用于投标人对投标文件内容的真实性、准确性和知识产权合规性作出正式声明。

### 模板正文

```
关于投标文件的声明函

致：{招标代理机构全称}

关于贵方编号为{招标编号}的投标邀请，本单位愿意参加{项目全称}的投标，并已在本单位此次递交的投标文件中（正本一份，副本{N}份）按照招标文件中要求的内容和格式充分、如实、准确地向贵方提供所有资料和信息。

在此，本单位正式声明：
对所有上述资料和信息的真实性和准确性负责，一旦发生上述资料和信息的失实和错误，将承担由此造成的一切责任。对项目所涉及的所有产品及服务的知识产权纠纷，全部责任由我司承担。



投标人授权代表姓名（签字或盖章）：
投标人名称：{公司全称}（公章）
日期：{年} 年 {月} 月 {日} 日
```

## 二、服务承诺函

服务承诺函表达投标人对招标文件技术需求和合同条款的全面理解与接受，以及对不分包不转包等核心要求的承诺。

### 模板正文

```
服务承诺

致：{采购人全称}

本公司郑重声明：
我公司已充分理解招标文件第三章"合同条款"和第五章"技术需求书"中的全部内容，保证满足上述章节中对投标人的各项要求以及{采购人简称}对供应商的管理要求。即，本公司同时满足以下条件：

1. 我公司有能力且愿意为招标人提供产品或服务，并接受招标人的监督管理。
2. 我公司保证本项目不进行分包和转包。
3. 服务标准符合国家服务标准或行业服务标准要求。


投标人授权代表姓名（签字或盖章）
投标人名称：{公司全称}（公章）
日期：{年} 年 {月} 月 {日} 日
```

## 三、采购纪律合规承诺函

用于承诺遵守采购纪律、合规经营、保密义务等，是政府采购和国企采购项目的常见要求。

### 模板正文

```
承诺函

致：{采购人全称}

本公司郑重声明：

一、本公司承诺遵守{采购人简称}采购纪律和相关工作流程，确保采购活动规范与廉洁。

二、本公司承诺依法成立，具有法人资格或具备独立承担民事责任的能力及有履行合同所必需的设备和专业技术能力。

三、本公司承诺商业信誉及经营状况良好，过去3年内经营活动中未出现重大违法违规等不良记录。

四、本公司保证提交的资料、数据（包括但不限于响应文件及其它要求提供的资料等）具有合法性和有效性。本公司所有提交的资料均视为本公司真实意思表达，对本公司具有法律约束力。

五、本公司严守采购活动中涉及的保密信息，未经法律法规许可要求或未经事先书面同意，绝不将信息泄露给其他第三方。

六、本公司承诺参与采购活动中履行以下义务：
（一）自觉遵守关于供应商管理的相关制度规定，保证在采购活动中无任何违规、违纪行为。
（二）提供符合规定质量标准的货物、工程和服务，未经事先书面同意，不擅自将中标（成交）项目分包或转包。
（三）在参与采购项目过程中，不串通其他供应商哄抬标价，扰乱采购秩序，谋取非法权益，不提供虚假材料谋取中标（成交）。
（四）中标（成交）后，按规定的时间签订采购合同，并按合同严格履约，不擅自变更或者终止采购合同。
（五）主动接受有关监督检查部门的监督、管理，不进行包括给回扣、介绍费在内的一切违纪违法行为。
（六）发生工商变更、注销，应书面或通过邮件报告，并办理相应的公司资格变更或注销手续。
（七）相关法律法规及规章制度规定的其他义务。

七、本承诺函由本公司法定代表人或授权代表签字并加盖公章后生效。

以上条款，若有违反任意一条，本公司自愿承担全部责任并接受相应处理。


投标人法定代表人或授权代表姓名（签字或盖章）：
投标人名称：{公司全称}（公章）
```

## 四、商业信誉及无违法记录承诺函

```
商业信誉承诺函

致：{采购人全称}

我公司作为{项目全称}投标人，在此郑重承诺：我公司具有良好的商业信誉和财务状况，过去3年内经营活动中未出现重大违法违规等不良记录。不分包、不转包。


投标人授权代表姓名（签字或盖章）：
投标人名称：{公司全称}（公章）
日期：{年} 年 {月} 月 {日} 日
```

## 五、其他服务承诺函

针对特定项目的额外服务承诺，内容可根据项目需要灵活调整。

```
其他服务承诺函

致：{采购人全称}

本公司郑重承诺：
1. 负责完成此系统测试及生产环境的软硬件的部署安装，包括数据库与中间件。如变更工作地点，免费提供移机服务。
2. 协助招标方完成相关网络备案工作。
3. 配合完成三级等保测评整改工作。
4. 系统开发过程中所涉及的客户资料、技术文档和程序源代码等，本公司严格保密。如中标按贵公司规定签订保密协议和保密承诺书，同时所有派驻项目组成员签保密承诺书。
5. 对于定制开发部分，提供全部源代码和相关文档。


投标人法定代表人或授权代表姓名（签字或盖章）：
投标人名称：{公司全称}（公章）
```

## 六、使用说明

1. 所有声明函和承诺函均需加盖投标人公章
2. 由法定代表人或授权代表签字
3. 日期应与投标文件其他部分保持一致
4. 如招标文件提供了固定格式，应优先使用招标方指定格式
5. 承诺内容不得与招标文件要求矛盾
6. 可根据项目需要增加或调整承诺条款

> 声明函和承诺函是投标文件合规性的重要组成部分，缺失可能导致废标。
