# bid-generator

投资管理领域标书生成工具，提供从招标文件解析到标书生成的全流程支持。

## 功能概览

| Skill | 说明 | 触发命令 |
|-------|------|----------|
| bid.parse | 解析招标文件，提取评分体系与需求清单 | `/bid.parse <文件路径>` |
| bid.plan | 生成标书骨架与章节结构 | `/bid.plan` |
| bid.fill | 按需填充章节正文（支持单章/批量） | `/bid.fill <章节名>` 或 `/bid.fill all` |
| bid.review | 五维质量审查（评分覆盖度、需求对应等） | `/bid.review` |

## 使用流程

```
bid.parse → bid.plan → bid.fill → bid.review
```

1. **解析**：提供招标文件（.docx/.doc/.pdf），提取评分标准和需求清单
2. **规划**：基于解析结果生成标书骨架大纲
3. **填充**：逐章或批量生成标书正文内容
4. **审查**：以评审专家视角进行质量审查

## 知识库

内置投资管理领域知识库，覆盖需求分析、总体方案、功能模块、技术方案、实施方案、维保方案、培训售后、公司资质、商务报价、程序资格文件等 10 大类。
