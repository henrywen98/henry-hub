## 背景: skill 是什么、它要解决什么问题

业务目标: **把核心技术拆成最小可独立定价 / 评估 / 复用的业务单元**。
- 客户问"这个能力多少钱、能不能给下个客户用" → 看能力清单层
- PM 问"模块到哪了、谁在做" → 看业务模块层
- 测试问"AI 抽得准不准" → 看 AI 字段层

输入异构(PRD/方案/报价),输出统一是 Excel 跟踪表,**手填列重跑不能被覆盖**是核心契约。
