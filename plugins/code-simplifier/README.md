# code-simplifier

代码简化与精炼代理。自动分析近期修改的代码，在保留全部功能的前提下，提升代码的清晰度、一致性和可维护性。

## 包含内容

### Agents

- **code-simplifier** - 代码简化专家代理（使用 Opus 模型），自动对近期修改的代码进行精炼优化

### 核心能力

- 保留功能：只改善代码写法，不改变代码行为
- 应用项目规范：遵循 CLAUDE.md 中的编码标准（ES modules、函数声明、类型注解等）
- 提升清晰度：减少不必要的复杂度和嵌套，消除冗余代码，改善命名
- 保持平衡：避免过度简化，不牺牲可读性换取行数更少

## 安装

```
/plugins install code-simplifier@henry-hub
```

## 使用示例

安装后，代理会在代码编写或修改后自动运行，无需手动触发。

```
# 代理会自动检测近期修改的代码并进行优化，例如：
# - 将嵌套三元运算符改为 switch/if-else
# - 消除冗余抽象
# - 统一命名规范
# - 移除描述显而易见逻辑的注释
```

如需对更大范围的代码进行审查，可明确指示代理扩大审查范围。
