# SVG Diagram Generator

生成专业、简约风格的 SVG 图表，完整支持中文文本。

## 功能

- **流程图** — 自上而下或从左到右的流程展示
- **架构图** — 分层布局：客户端 → 服务层 → 数据层
- **系统拓扑图** — 节点与连线展示数据流和依赖关系
- **时序图 / 泳道图** — 生命线、消息箭头、激活框

## 使用方式

安装后，当你请求画图时会自动触发：

- "帮我画一个流程图"
- "画个架构图"
- "系统拓扑图"
- "visualize this process"

## 设计原则

- 极简风格：细线条、大留白、柔和色彩，无渐变无阴影
- 严格对齐：所有元素对齐到 20px 网格
- 中文支持：使用 PingFang SC / Microsoft YaHei / Noto Sans SC 字体栈
- 自包含：所有样式、标记、定义均内联，无外部依赖

## 安装

```
/plugins install svg-diagram@henry-hub
```
