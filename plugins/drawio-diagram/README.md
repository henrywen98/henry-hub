# drawio-diagram

Draw.io 图表生成器 — 流程图、架构图、系统拓扑图、业务流程全景图，支持自动核验和复杂流程分区。

## 功能特色

- **通用图表**: 流程图、架构图、系统拓扑、时序图、业务全景图
- **三种输入模式**: 从代码分析提取业务逻辑 / 从需求文档生成 / 用户直接描述
- **自动核验**: 生成后自动导出 PNG 截图，AI 视觉检查布局、文字、连线
- **复杂流程分区**: 节点 > 30 或模块 > 5 时自动拆分为多 page，含全景总览
- **逻辑断点检测**: 从代码/文档分析时自动发现孤立事件、缺失反向操作、手动瓶颈等
- **中文原生支持**: draw.io 默认支持 CJK，无需特殊字体配置

## 输出格式

`.drawio` — 可用 draw.io 桌面版、VS Code 扩展、或 https://app.diagrams.net 打开和编辑。
