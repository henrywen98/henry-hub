# GitHub Issue 生成器

将原始信息整理成详细的 GitHub Issue 格式，包括标题、标签、详细描述、复现步骤等。

## 使用场景

- 将 Bug 报告转化为标准 GitHub Issue
- 从需求描述生成 Feature Request Issue
- 批量生成格式规范的 Issue

## 安装

```
/plugins install github-issue-generator@henry-hub
```

## 触发短语

- "生成 Issue"
- "创建 Issue"
- "生成 GitHub Issue"
- "Issue 格式整理"
- "创建 GitHub 问题"

## 使用示例

提供 Bug 描述或需求信息，系统将自动生成包含以下内容的 GitHub Issue：
- 标题（简洁明确）
- 标签（bug / feature / enhancement 等）
- 详细描述
- 复现步骤（如适用）
- 预期行为与实际行为
