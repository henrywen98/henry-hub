## 问题描述

### 背景
客户反馈订单模块存在多个问题需要处理

### 详细描述

#### 问题1：订单提交失败
- **问题类型**: Bug
- **详细描述**: 订单模块经常无法提交，提示"系统繁忙"
- **期望行为**: 订单能够正常提交
- **实际行为**: 提交时提示"系统繁忙"，无法完成提交

#### 问题2：批量导出功能
- **问题类型**: Feature
- **详细描述**: 目前订单只能逐个导出，效率低
- **期望行为**: 支持批量导出多个订单
- **实际行为**: 只能逐个导出

#### 问题3：审批流程简化
- **问题类型**: Enhancement
- **详细描述**: 审批流程太复杂，步骤过多
- **期望行为**: 简化审批流程，减少步骤
- **实际行为**: 审批步骤过多，用户体验差

#### 问题4：登录页面性能
- **问题类型**: Bug
- **详细描述**: 登录页面加载慢
- **期望行为**: 登录页面快速加载
- **实际行为**: 页面加载时间过长
