# pptx

演示文稿（.pptx）创建、编辑和分析工具。支持从零创建精美演示文稿、基于模板生成、编辑现有文件、处理布局和备注等全方位演示文稿任务。

## 使用场景

当用户需要进行以下操作时触发：

- 创建新的 PowerPoint 演示文稿
- 修改或编辑现有 .pptx 文件
- 使用模板创建演示文稿
- 处理幻灯片布局和设计
- 添加批注或演讲备注
- 将演示文稿转换为图片进行分析

### 支持的工作流

- **从零创建**：使用 html2pptx 将 HTML 转换为 PowerPoint，支持精确定位
- **基于模板创建**：分析模板、复制布局、替换内容
- **编辑现有文件**：解包 OOXML、编辑 XML、重新打包
- **可视化验证**：生成缩略图网格进行布局检查

## 安装

```bash
/plugins install pptx@henry-hub
```
