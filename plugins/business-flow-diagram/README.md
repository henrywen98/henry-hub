# business-flow-diagram

从代码库中分析业务逻辑，自动生成 Excalidraw 业务流程全景图。

## 功能

- 支持两种输入源：**代码库**分析 或 **需求文档**(PRD/业务说明书)解读
- 自动分析状态机、事件链、审批流程等业务逻辑
- 生成业务人员友好的流程图（纯业务语言，无代码术语）
- 自动识别逻辑断点（缺失的自动化、未订阅的事件等）
- 输出标准 `.excalidraw` 文件，可在 excalidraw.com 中打开并导出为 PNG/SVG
- 使用并行子代理策略高效探索大型代码库
- 内置自动化验证（重叠检测、编码校验、术语过滤）

## 使用方式

在 Claude Code 中对任意项目说：

```
画一个业务流程全景图
```

或者更具体地：

```
分析这个系统的业务逻辑，生成 Excalidraw 流程图，帮我发现逻辑断点
```

## 输出说明

生成的 `.excalidraw` 文件包含：

1. **模块区域** — 按业务模块分区（颜色编码）
2. **状态流转** — 每个模块的状态机可视化
3. **系统自动联动** — 跨模块自动化流程列表
4. **待完善功能** — 已识别的逻辑断点（红色标注）

## 打开方式

1. 访问 [excalidraw.com](https://excalidraw.com)
2. 左上角菜单 → Open → 选择生成的 `.excalidraw` 文件
3. 导出为 PNG/SVG：菜单 → Export image
