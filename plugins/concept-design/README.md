# concept-design — 概要设计说明书生成器

面向甲方交付的软件项目概要设计说明书自动生成工具。

## 功能

扫描项目中的 PRD 和技术架构设计文档，按固定7章模板结构，自动生成完整的概要设计说明书（`概要设计说明书.md`）。

## 模板结构

| 章节 | 内容 | 数据来源 |
|------|------|---------|
| 第1章 引言 | 编写目的、读者对象、术语表、参考资料 | 固定模板 + 项目元信息 |
| 第2章 技术选型 | 硬件环境、体系结构 | 技术架构设计 |
| **第3章 功能模块设计** | **模块全景图、功能设计表格、流程图、汇总表** | **PRD（核心，占60%+）** |
| 第4章 非业务功能需求 | 安全性、稳定性、性能 | 技术架构设计 |
| 第5章 系统集成方案 | 架构、部署、技术路线、界面设计 | 技术架构设计 |
| 第6章 接口设计 | 接口规范、模块接口概述 | 接口设计文档 |
| 第7章 安全性设计 | 风险、安全层次、策略、备份 | 技术架构设计 |

## 输入要求

- **必需**：PRD（产品需求文档）、技术架构设计文档
- **可选**：数据库设计文档、接口设计文档

## 使用方式

```
生成概设
写概要设计
概要设计说明书
```

- Phase 4.5 增加名称对齐检查、共享模块声明核验、copy-paste 错误检测
- 模板 template.md 更新全景图示例和第5章去重指导
