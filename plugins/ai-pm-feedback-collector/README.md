# AI 产品经理反馈收集器

将客户反馈快速整理成结构化文档，包括 Bug 描述、优化建议、优先级排序。

## 使用场景

- 整理客户反馈为结构化需求文档
- 将聊天记录或语音反馈转成结构化文本
- 生成 PRD 需求说明
- 梳理问题优先级

## 安装

```
/plugins install ai-pm-feedback-collector@henry-hub
```

## 触发短语

- "整理客户反馈"
- "生成需求文档"
- "梳理问题优先级"
- "客户反馈整理"
- "问题整理"

## 使用示例

将客户反馈原文粘贴给 Claude，系统将自动整理为包含以下内容的结构化文档：
- Bug 描述与复现步骤
- 优化建议与预期效果
- 优先级排序与推荐处理顺序
