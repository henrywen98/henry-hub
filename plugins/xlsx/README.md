# xlsx

全面的电子表格（.xlsx、.xlsm、.csv、.tsv 等）创建、编辑和分析工具。支持公式、格式化、数据分析和可视化，内置财务模型的行业标准规范。

## 使用场景

当用户需要进行以下操作时触发：

- 创建新的电子表格（含公式和格式化）
- 读取或分析电子表格数据
- 修改现有电子表格（保留公式）
- 数据分析和可视化
- 重新计算公式
- 构建财务模型

### 支持的工具链

- **pandas**：数据分析、批量操作、数据导出
- **openpyxl**：复杂格式化、公式、Excel 特定功能
- **LibreOffice**：公式重新计算（通过 recalc.py 脚本）

### 内置规范

- 财务模型颜色编码标准（蓝色=输入、黑色=公式、绿色=跨表引用）
- 数字格式化规范（货币、百分比、倍数等）
- 公式构建规则和错误预防
- 零公式错误的质量要求
