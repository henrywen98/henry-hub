# 需求确认单生成器

将各类需求素材（原型图、会议纪要、字段说明、业务逻辑、调研记录等）统一整理为标准需求确认单格式。

## 使用场景

- 整理客户反馈为正式需求文档
- 将会议纪要转化为需求确认单
- 将零散的需求笔记整理成规范文档
- 从原型图/UI截图提取需求描述

## 安装

```
/plugins install gen-requirement-doc@henry-hub
```

## 适用输入

- 原型图 / UI 截图
- 会议纪要 / 调研记录
- 字段清单 / 数据字典
- 业务逻辑描述
- 零散的需求笔记
- 客户反馈邮件或聊天记录

## 使用示例

提供需求素材后，系统将自动：
1. 收集并确认需求主题、类型、素材来源
2. 按标准模板整理为需求确认单
3. 输出包含功能描述、验收标准等的完整文档
